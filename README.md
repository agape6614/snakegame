# Snake Game (Pygame)

Pygame을 이용해 구현한 간단한 Snake 게임입니다.  
방향키 대신 숫자 키패드를 사용하여 조작할 수 있으며, 먹이를 먹을수록 속도가 증가합니다.

---

## 게임 소개

- 기본적인 Snake 게임 구현
- 먹이를 먹으면 길이 증가 및 속도 증가
- 벽 또는 자기 몸에 부딪히면 게임 초기화
- 점수(길이) 화면에 표시

---

## 조작 방법

| 키 | 동작 |
|----|------|
| 8 (키패드) | 위 |
| 5 (키패드) | 아래 |
| 4 (키패드) | 왼쪽 |
| 6 (키패드) | 오른쪽 |
| Q | 게임 초기화 |
| 창 닫기 | 종료 |
